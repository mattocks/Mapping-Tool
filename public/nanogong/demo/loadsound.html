The NanoGong Applet with a Recorded Voice

The NanoGong applet below loads a pre-recorded voice file from a website. The location of the voice file is given by an applet parameter named "SoundFileURL". In this particular case, the value of the parameter is the relative location of the file in the folder.

More information about NanoGong can be obtained from http://gong.ust.hk/nanogong/.
